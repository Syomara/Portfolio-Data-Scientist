# Power BI Dashboards

En esta carpeta se incluyen los materiales del proyecto realizado en **Power BI**, con un enfoque en el análisis de datos y la presentación de conclusiones.

## 📑 Contenido
- 📄 PDF con los datos y conclusiones del proyecto
- 🖼️ Imágenes de las visualizaciones creadas en Power BI
- 🔗 [Acceso al archivo original en Google Drive](https://drive.google.com/file/d/1jro4eEkuvMmLf6ixPqnsm2cI_1ukZbwN/view?usp=sharing)

## 🎯 Objetivo
El proyecto busca mostrar cómo transformar datos en información clara y visual, facilitando la toma de decisiones estratégicas mediante dashboards interactivos.
